Rovo Dev CLI 介绍

Atlassian 推出了 Rovo Dev CLI，这是一款功能强大的人工智能代理，旨在无缝集成到您的开发终端中。

Rovo Dev CLI 是什么？

Rovo Dev 是 Atlassian 为软件开发者构建的 AI 代理，现在可以通过命令行界面 (CLI) 直接在您的终端中使用。这种集成将 Rovo Dev 的自然语言能力带入开发者最熟悉的环境之一，使您无需离开命令行即可实现更快、更智能的编码。

连接 Jira、Confluence 等更多应用

Rovo Dev 与 Atlassian 应用程序无缝集成，使开发者能够利用 Atlassian 工具的全套功能进行项目管理、协作和版本控制。通过 Rovo Dev CLI，您可以直接在终端中完成 Jira 工作项、轻松更新 Confluence 文档、维护文档、调试和理解复杂的代码库等等。Atlassian 生态系统的这一进步通过提供一个内聚的环境，让开发者能够简化工作流程并保持生产力，而无需切换上下文。

为什么使用 Rovo Dev CLI？

  • 🚀 通过 AI 辅助提高您的生产力
  • 💻 保持在您的终端工作流中
  • 🤝 通过 MCP 服务器连接您现有的工具
  • 📚 提高代码质量和文档
  • ⏱️ 节省日常编码任务的时间

功能概述：

  • 代码生成：获取 AI 辅助编写新的代码片段和函数
  • 代码理解：通过自然语言解释快速理解复杂的代码块
  • 代码审查：接收自动代码审查建议和改进
  • 文档帮助：生成和改进代码文档

安装 Rovo Dev CLI

Rovo Dev 作为 Atlassian 命令行界面 (ACLI) 的扩展提供。要安装 Rovo Dev：

  1. 安装 ACLI（如果您已安装 ACLI，请更新到最新版本），请根据您的操作系统使用以下说明：
  • macOS
  • Linux
  • Windows
  1. 前往您的 Atlassian 个人资料，通过选择 创建 API 令牌 来创建无范围的 API 令牌。
  2. 在您的终端中，运行以下命令以使用您的 Atlassian 帐户进行授权：
acli rovodev auth login
  1. 运行此命令以交互模式 (IM) 打开 Rovo Dev：
acli rovodev run

激活或停用 Rovo Dev CLI

您需要 Rovo Dev Agents 管理员权限才能开启和关闭 Rovo Dev CLI。Rovo Dev CLI 作为 Rovo Dev Agents 测试版的一部分自动启用。

设置生效可能需要 1-3 个工作日。

目前禁用 Rovo Dev CLI 将影响整个组织对 Rovo Dev CLI 的访问，我们仍在努力实现站点级别控制。

一旦您注册 Rovo Dev CLI，您将在我们的文档中看到详细说明。

如何获得支持：
